O que é HTML?

O HTML (Linguagem de Marcação de Hipertexto) é uma linguagem de programação usada para criar páginas na web e o conteúdo que há nelas, com o auxílio de tags.

As tags

As quatro tags básicas são:

* h1 - É a tag básica para criar cabeçalhos. Vai até o "h6".

* p - Serve para criar parágrafos.

* i ou em - Deixa o texto em itálico.

* strong - Deixa o texto selecionado em negrito.

CSS

O CSS (Cascading Style Sheets), ou em Português, Folhas de Estilo em Cascatas, é uma forma de adicionar estilo a um código HTML com tags que não estão disponíveis no HTML. As tags do CSS podem ser usadas para adicionar uma margem, mudar a cor do fundo, a cor da fonte, entre outras coisas.
